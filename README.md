
# Ez-Fx

A free audio effects rack / pedalboard in your browser.


## Features

- 15 audio effects
- Log in and save / load effects presets
- Fullscreen mode 
- Light/dark mode 


## Installation

Install dependencies with npm

```bash
  npm install 
```
    
Start local server

```bash
  npm run dev
```
